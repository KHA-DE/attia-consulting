The primary action control. Red fill for the main action; outline or ghost for secondary; dark for inverse contexts.

```jsx
<Button>Termin vereinbaren</Button>
<Button variant="outline">Mehr erfahren</Button>
<Button variant="ghost" size="sm">Details →</Button>
```

Variants: `primary` (red), `dark` (ink), `outline`, `ghost`. Sizes: `sm` `md` `lg`.
